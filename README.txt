Requerimientos
==============

 - Para ejecutar este programa se necesita tener instalado
   Python 3.x y los paquetes getpass, pycrypto y binascii


Instrucciones
=============

 - Para correr el programa, ejecute en este directorio

   	$ python src/ssss.py

   Esto imprimira un mensaje con las intrucciones para la
   ejecucion del programa

 - Para ejecutar las pruebas unitarias, ejecute en este directorio

   	$ python tests/test_suite.py
