def ssss.check_args_syntax (   args)

Revisa si los argumentos del programa cumplen la sintaxis requerida.

Revisa si las banderas son correctas, si los archivos existen y si los numeros de claves necesarias y generadas son validos. De no ser asi, imprime un mensaje de ayuda y termina la ejecucion.

Parámetros
argsargumentos que se le proporcionaron al programa
Devuelve
1 si los argumentos cumplen con la sintaxis para cifrar un archivo, 2 si cumplen con la sintaxis para descifrar un archivo

def ssss.pairs_file_syntax (   f)

Revisa si el archivo con las claves para descifrar cumple con la sintaxis requerida.

La sintaxis que debe tener cada linea del archivo con las claves es 'x,f(x)', sin comillas, donde x y f(x) son enteros no negativos.

Parámetros
fel archivo que contiene las claves para desencriptar
Devuelve
True si el archivo cumple el formato, False en otro caso
